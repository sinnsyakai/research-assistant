# リサーチ＆要約アシスタント 使い方ガイド

## 🎯 このアプリでできること

AIが**信頼できる情報源**から記事や論文を検索し、**日本語で要約**してくれます。

---

## 🚀 基本的な使い方

### Step 1: APIキーを設定
右上の**歯車アイコン**をクリックして、以下のキーを入力：
- Google Search API Key
- Google CSE ID  
- Gemini API Key

### Step 2: 検索モードを選択

| モード | 説明 |
|-------|------|
| 📰 **ニュース** | 日本の主要メディアから検索 |
| 🌐 **海外の情報** | 海外メディア（NYT、BBC等）から検索 |
| 📚 **論文** | 学術論文を検索 |

### Step 3: 検索キーワードを入力
例：
- 「睡眠障害」
- 「SNSの最新情報」
- 「子どもの学力」

### Step 4: 結果から記事を選択
チェックボックスで記事を選択

### Step 5: 要約を生成
**「選択した記事を要約」**ボタンをクリック

---

## 📝 要約スタイル

| スタイル | 用途 |
|---------|------|
| **概要** | 全体像をサッと把握（1000文字以内） |
| **記事形式** | ブログ記事として読む（2000〜5000文字） |
| **詳細** | 深く学習する（5000文字以上） |

---

## 💡 検索のコツ

### ✅ 良い検索例
- 「SNSの最新情報」→ TikTok、Instagram等のニュースを検索
- 「睡眠障害 治療」→ 不眠症、睡眠時無呼吸症等の論文を検索
- 「スイーツ トレンド」→ 流行のデザート情報を検索

### ❌ 避けるべき検索
- 極端に短いキーワード（「ニュース」だけ）
- 固有名詞のみ（より具体的に）

---

## 🔍 機能詳細

### AI翻訳機能
- **海外の情報モード**: 日本語で検索 → 英語に翻訳して検索 → 結果を日本語に翻訳
- **論文モード**: 英語論文のタイトル・要約を日本語化

### 日付フィルタ
「最新」「今週」などのキーワードで自動的に期間を絞り込み

### 品質フィルタ
- 掲示板、個人ブログ、PR記事を自動除外
- AIが関連性を判定して無関係な結果を除外

---

## ❓ よくある質問

**Q: 海外の情報が表示されない**
A: 「海外の情報」モードを選択しているか確認してください

**Q: 翻訳されない**
A: Gemini APIキーが正しく設定されているか確認してください

**Q: 論文が古い**
A: 2020年以降の論文を優先表示しています
